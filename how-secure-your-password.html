Indirectly question is "How secure you are?", means password is key of your lock and if key itself not safe then how you? So password matters all about your privacy and security.       Strong password has certain properties like minimum 8 char length, alphanumerical, special symbol. So for now consider your password has all these properties, still it may be weak password and how? Let see,


      1. Common password

What is common password? I know you listened this word before. Answer is most used password such as password, user, 123456789, admin@123, Unknown as password. Lets go in deep. Common password are stolen from different sources such as suppose someone hack Facebook database, Microsoft database or may even other small level website. So that he got bunch of password and you password may belong to that list. So to safe your password from common password threats, your password should be unique.

3. Phishing

Even you password is strong but what if you become victim of phishing. So phishing is another way that make your password weak. So always stay alert from phishing.
